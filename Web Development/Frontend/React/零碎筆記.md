### 為何 `componentDidMount()` 被呼叫兩次？

在 developement mode 時，因為 `index.tsx` 使用了 `<React.StrictMode>` 所以才會出現 lifecycle hook 被呼叫兩次的狀況，當運行 production mode 的程式碼時就不會有此現象了。如果在開發階段也不想看到，就必須將 `<React.StrictMode>` 移除。
