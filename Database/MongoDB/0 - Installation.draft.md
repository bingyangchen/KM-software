# 安裝

請參考 [官方網站](https://www.mongodb.com/docs/manual/installation/)

# 啟動 MongoDB Server

```bash
# Start as a MacOS service
brew services start mongodb-community@6.0
```

# 關閉 MongoDB Server

```bash
brew services stop mongodb-community@6.0
```
